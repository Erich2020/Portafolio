La Consentida

El negocio realizaba registro y control de sus ventas, inventario de forma tradicional, se trabajó en sus principales procesos, de los cuales se tomaron para el desarrollo de un software punto de venta, adicional se contempló la expansión del negocio en otros lugares, requiriendo la facilidad de monitorear a la distancia los inventarios y sus ventas por negocio.
Se Desarrolló el sistema en 3 Fases:

  • Fase 1: Desarrollo de software punto de venta.
    Desarrollado en Tecnología .Net 4.5 con CSharp
    Conectado a Bases de Datos embebidos mediante Firebird, y a los servicios web.
    Genera Documentos en Formato PDF (ventas, gestión de Inventario, cortes de caja.).
    Envió de tickets, productos faltantes por medio de Whats App.
    Multi-usuario y roles.
  • Fase 2: Desarrollo de Servicios Web y Web.
    Desarrollado en Tecnología .Net 5 con CSharp
    Conectado a Bases de Datos mediante SQL server Express.
    Control y Autenticación por medio de JWT
    Página web con DashBoard.
  • Fase 3: Desarrollo de Aplicación Móvil.
    Desarrollado con Java utilizando el SDK Android.
    Comunicación con los servicios web para la gestión de inventario e inventarios cíclicos.
